# dotfiles

Personal dotfiles managed with [stow](https://www.gnu.org/software/stow/).

## Dependencies

```sh
brew install stow fzf eza starship zoxide
```

> On Linux, substitute `brew` with your distro's package manager (`apt`, `pacman`, etc.), or install [Homebrew for Linux](https://docs.brew.sh/Homebrew-on-Linux).

## Setup

```sh
brew install stow
stow nvim
stow zsh
stow wezterm
```
